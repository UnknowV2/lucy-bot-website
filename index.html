Fitur Andalan Lucy Bot

Sistem Leveling Aktif

Tingkatkan interaksi server dengan sistem XP otomatis. Anggota akan naik level saat aktif berkirim pesan, lengkap dengan notifikasi level-up.

Sistem Ekonomi & Game

Sistem koin dengan hadiah harian, transfer, dan leaderboard. Mainkan game seru seperti slot, adu kartu, dan dadu!

Moderasi Profesional

Kelola server Anda dengan mudah menggunakan perintah kick, role permanen, dan role berjangka waktu.

Logging Canggih

Sistem log via Webhook otomatis untuk semua aktivitas server: pesan, member, channel, role, dan lainnya.

Sistem Tiket & Register

Panel dukungan tiket pribadi dan sistem registrasi member baru untuk menjaga server tetap terorganisir dan aman.

Dashboard Real-Time

Pantau statistik bot seperti uptime, latensi, penggunaan memori, dan daftar server secara langsung melalui web.

Daftar Perintah

Berikut adalah beberapa perintah utama yang bisa Anda gunakan. Untuk daftar lengkap, gunakan perintah /help di server Anda.

Ekonomi & Game

  • /cash - Mengecek saldo koin Anda.
  • /daily - Mengklaim hadiah koin harian.
  • /give - Memberikan koin ke pengguna lain.
  • /leaderboard - Menampilkan papan peringkat.
  • !spin [bet] - Memutar mesin slot.
  • !card [bet] - Bermain tebak kartu tertinggi.
  • !dadu [bet] - Bermain adu dadu.

Moderasi

  • /kick - Mengeluarkan anggota dari server.
  • /purge - Menghapus pesan dalam jumlah besar.
  • /give-role - Memberikan peran permanen.
  • /remove-role - Menghapus peran dari anggota.
  • /role-time - Memberi peran berjangka waktu.
  • /announcement - Mengirim pengumuman.

Konfigurasi (Admin)

  • /set-logs - Mengatur channel log server.
  • /set-join-leave - Mengatur log anggota keluar/masuk.
  • /set-ticket - Mengatur panel sistem tiket.
  • /set-autorole - Mengatur peran otomatis.
  • /set-leveling-logs - Mengatur log level-up.
  • /giveaway start - Memulai giveaway.
